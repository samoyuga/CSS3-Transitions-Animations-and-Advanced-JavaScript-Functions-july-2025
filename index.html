Explore the power of combining CSS animations with JavaScript interactivity

Animation Controls

Use JavaScript to control CSS animations dynamically

// JavaScript function to add CSS class
function startAnimation(animationType) {
  const box = document.getElementById('animated-box');
  box.classList.add(animationType);
}

Interactive Animation

Control animation properties with JavaScript

// Change animation duration with JS
function changeAnimationDuration(duration) {
  const box = document.getElementById('interactive-box');
  box.style.animationDuration = `${duration}s`;
}

Function Scope Demo

Demonstrating local and global scope in JavaScript

// Global variable
let animationCount = 0;

function updateAnimationCount() {
  // Local variable with same name
  let animationCount = 5;
  animationCount++; // Affects local variable
  console.log("Local count:", animationCount);
}

Animation Toggles

Toggle animations with classList methods

// Toggle animation with classList
function toggleAnimation() {
  const box = document.getElementById('toggle-box');
  box.classList.toggle('spin');
  box.classList.toggle('color-change');
}
